## 🚀 Sobre mí

Soy desarrollador Full Stack con más de 4 años de experiencia creando aplicaciones web, sistemas empresariales ERP e integraciones backend.

Actualmente desarrollo y mantengo soluciones para procesos empresariales críticos, incluyendo módulos ERP, APIs, automatización de procesos e integraciones con sistemas externos.

Me enfoco en construir software mantenible, escalable y alineado con las necesidades del negocio. Me apasionan el desarrollo backend, la arquitectura de software, el diseño de sistemas escalables y la creación de productos SaaS.
